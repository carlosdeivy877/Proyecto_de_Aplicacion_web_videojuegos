Proyecto de Aplicación Web: Videojuegos

Aplicación Web de Videojuegos

Este proyecto fue desarrollado como parte de las actividades académicas de la asignatura de Desarrollo Web. El objetivo principal fue crear una aplicación web interactiva que permitiera poner en práctica los conocimientos adquiridos durante el curso, integrando tecnologías de frontend y backend.

La página presenta una selección de videojuegos favoritos, incorpora un sistema de comentarios y permite al usuario cambiar entre modo claro y modo oscuro para mejorar la experiencia de navegación.

 Funcionalidades

* Visualización de videojuegos con imágenes y descripciones.
* Diseño adaptable a computadores, tabletas y dispositivos móviles.
* Cambio entre tema claro y oscuro.
* Formulario para registrar comentarios.
* Visualización dinámica de comentarios.
* Servidor local desarrollado con Node.js y Express.

 Tecnologías utilizadas

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* Node.js
* Express.js
* Git y GitHub

Ejecución local

1. Clona el repositorio:

```bash
git clone https://github.com/carlosdeivy877/Proyecto_de_Aplicacion_web_videojuegos.git
```

2. Accede a la carpeta del proyecto:

```bash
cd Proyecto_de_Aplicacion_web_videojuegos
```

3. Instala las dependencias:

```bash
npm install
```

4. Inicia el servidor:

```bash
node server.js
```

5. Abre el navegador y visita:

```text
http://localhost:3000
```
